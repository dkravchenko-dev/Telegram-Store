## 📖 Описание

**Telegram Store** — это полнофункциональный Telegram-бот для автоматизации продажи цифровых товаров. Бот позволяет организовать магазин конфигураций, подписок и других цифровых продуктов с интегрированной платежной системой, административной панелью и системой управления пользователями.

### 🎯 Основные преимущества

- ✅ **Полная автоматизация** продаж цифровых товаров
- ✅ **Встроенная платежная система** для приема оплаты
- ✅ **Административная панель** для управления товарами и пользователями
- ✅ **Система подписок** с автоматическим продлением
- ✅ **Безопасность** — блокировка мошенников и контроль доступа
- ✅ **Простота использования** — интуитивный интерфейс для пользователей

## 📦 Установка

### Предварительные требования

- Python 3.11 или выше
- pip (менеджер пакетов Python)
- Git

### Шаги установки

1. **Клонируйте репозиторий**
   ```bash
   git clone <repository-url>
   cd Telegram-Store
   ```

2. **Создайте виртуальное окружение** (рекомендуется)
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate  # Linux/macOS
   # или
   .venv\Scripts\activate  # Windows
   ```

3. **Установите зависимости**
   
   Для **Linux**:
   ```bash
   pip install -r requirements_linux.txt
   ```
   
   Для **macOS**:
   ```bash
   pip install -r requirements_mac.txt
   ```

4. **Настройте конфигурацию** (см. раздел [Настройка](#-настройка))

## ⚙️ Настройка

### 1. Получение токена бота

1. Найдите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте полученный токен

### 2. Настройка конфигурации

Откройте файл `app/config.py` и заполните необходимые параметры:

```python
TOKEN_BOT_USERS = "YOUR_BOT_TOKEN_HERE"  # Токен вашего бота
ID_ADMIN = 123456789  # Ваш Telegram ID (можно узнать у @userinfobot)
SUPPORT_PROFILE = "@your_support_username"  # Username для поддержки
```

### 3. Настройка структуры данных

Убедитесь, что файл `database/users.json` существует и имеет правильную структуру:

```json
{}
```

Бот автоматически создаст необходимую структуру при первом запуске.

## 🚀 Использование

### Запуск бота

#### Простой запуск
```bash
python main.py
```

#### Запуск в фоновом режиме (Linux/macOS)
```bash
chmod +x start.sh
./start.sh
```

Скрипт `start.sh` автоматически:
- Останавливает предыдущий процесс бота
- Активирует виртуальное окружение
- Запускает бота в фоновом режиме
- Сохраняет логи в `output.log`

### Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Запуск бота и главное меню |

## 🔒 Безопасность

- ⚠️ **Никогда не публикуйте** файл `app/config.py` с реальными токенами
- ⚠️ Добавьте `app/config.py` в `.gitignore`
- ⚠️ Храните токены в переменных окружения или секретах
- ⚠️ Регулярно проверяйте логи на подозрительную активность
