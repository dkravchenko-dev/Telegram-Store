# Telegram Store

**telegram_store** — это Telegram-бот на Python с использованием [aiogram](https://docs.aiogram.dev/), который позволяет организовать магазин цифровых товаров. Бот предназначен для продажи конфигураций и других цифровых товаров за реальные деньги с разделением пользователей на обычных и администраторов

## Функциональность

### Обычные пользователи
- Ознакомление с сервисом и просмотр инструкции
- Просмотр каталога товаров
- Покупка нужного количества цифровых товаров
- Продление или докупка подписки при необходимости
- Управление своим профилем и история покупок

### Администратор
- Добавление, удаление и редактирование товаров
- Просмотр всех пользователей и их заказов
- Проверка достоверности оплаты пользователя
- Блокировка пользователя в один клик при нарушениях или мошенничестве
- Управление финансами магазина
- Отправка уведомлений пользователям

---

## Зачем нужен бот
- Автоматизация продаж цифровых товаров в Telegram
- Упрощение управления товарами и подписками
- Быстрый контроль оплаты и безопасность пользователей
- Разделение прав доступа между пользователями и администраторами

---

## Технологии
- Python 3.11+
- [aiogram](https://docs.aiogram.dev/) для работы с Telegram API
- Хранение данных всех пользователей в JSON-файле
- Собственная платежная система для оплаты цифровых товаров

---

## Установка и запуск

Склонируйте репозиторий и установите зависимости в зависимости от вашей системы. Пример для Linux и macOS:

```bash
git clone <репозиторий>
cd telegram_store

# Установка зависимостей
pip install -r requirements_linux.txt   # для Linux
pip install -r requirements_mac.txt     # для macOS

# Запуск бота
python main.py
